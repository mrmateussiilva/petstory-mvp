# Fontes do livro PDF

Coloque aqui arquivos de fonte `.ttf` para serem usados nas escritas do livro (capa, subtítulo e contracapa).

- O primeiro arquivo `.ttf` encontrado na pasta será usado.
- Exemplo: baixe uma fonte em [Google Fonts](https://fonts.google.com/) (ex.: Roboto, Open Sans), extraia o `.ttf` e salve nesta pasta.

Se não houver nenhum `.ttf`, o PDF usará a fonte padrão (Helvetica).
